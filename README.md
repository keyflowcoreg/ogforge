# OGForge

AI-powered OG image and social card generator.

[![Live Demo](https://img.shields.io/badge/Live_Demo-ogforge.vercel.app-blue?style=for-the-badge)](https://ogforge.vercel.app)

[Live Demo](https://ogforge.vercel.app) | [Report Bug](https://github.com/keyflowcoreg/ogforge/issues)

## Features

- Generate OG images, Twitter cards, LinkedIn banners, and Instagram stories
- 6 color presets and 5 styles (Gradient, Minimal, Bold, Dark Mode, Photo)
- Live editor with real-time preview
- Free tier with 1 watermarked image/day, Pro for unlimited clean exports
- x402 crypto payments (USDC on Base)

## Tech Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · @vercel/og · Framer Motion · x402 Payments

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4009](http://localhost:4009) to view it locally.
